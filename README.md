# Text to sql app

## Connects to a database and helps us to retreive data while wrting the query in plain english.

### Tech Stack:
    - Pyhton
    - Langchain
    - sqlite3
    - ollama
    - model: llama3.2:1b
    - streamlit

Ollama helps us to run the llm models in local machine
https://ollama.com/

To pull a model to local machine from the ollama registry

```
ollama pull llama3.2:1b
```
in place of llama3.2:1b, can replace the model we want
